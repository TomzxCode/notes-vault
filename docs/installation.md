# Installation

## Requirements

- Python 3.14 or newer
- [uv](https://docs.astral.sh/uv/) (recommended) or pip

## Install Notes Vault

```bash
uv tool install https://github.com/tomzxcode/notes-vault.git
```

## Verify Installation

```bash
nv --help
```

## Entry Points

Notes Vault installs two command aliases:

| Command | Description |
|---------|-------------|
| `nv` | Primary CLI |
| `notes-vault` | Alias for `nv` |
